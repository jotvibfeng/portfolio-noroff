Cross Course Project

The first project we got at Noroff was to make a website of three choice templates to pick from Gaming, Movie, and Outerwear

What purpose has the site ?

  • To create a user-friendly site where you can purchase games and also sell your used games.
  • You should have an easy time navigating around the site if you were to go on contact or finish your purchase at the site.
  • Be available to look in detail at the product to get more specific info about the product.

    • How to install the project ?

  • Clone the repo: git clone git@github.com:jotvibfeng/Project_Exam1.git
  • Install the dependencies: npm install
  • To run the app, run the following commands: npm run start
