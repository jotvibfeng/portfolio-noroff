Semester Project

We are half way through first year and now it was time to use what we could do so far by making a website for a new natrual of history museum that is soon to open.

What purpose has the site ?

  • It should appeal to families with small children and middle school students
  • The museum wanted to make learning fun and exciting for the audience.
  • Have an easy navigating site with good information about opening hours, what exhibitions there are on show and other informative details they need to know.

    • How to install the project ?

  • Clone the repo: git clone git@github.com:jotvibfeng/Project_Exam1.git
  • Install the dependencies: npm install
  • To run the app, run the following commands: npm run start
